# Project Overview

## プロジェクト概要
- 格闘ゲームのボタン練習を目的としたWindows用アプリケーションです。
- ランダムに表示されるお題に対し、対応するボタンを入力してスコアを競います。
- 常駐型で、ゲーム起動時や練習の合間に素早く利用できるよう設計されています。

- **`.vscode/settings.json`**: Visual Studio Codeエディタ固有の設定ファイルで、ワークスペースやプロジェクトに対する個別の設定（例えば、特定のファイルのフォーマット設定など）を定義します。

- **`tests/test_get_move_name_for_input.py`**: 特定の入力に対して対応するムーブ名が正しく取得されるかを検証するテストスクリプトです。
- **`tests/test_get_pressed_buttons.py`**: 物理的なコントローラー入力から押されたボタンを正確に検出できるかを検証するテストスクリプトです。
- **`tests/test_is_no_count_case.py`**: 特定の条件下でスコアがカウントされないロジックが正しく動作するかを検証するテストスクリプトです。
