# Project Overview

## プロジェクト概要
- 格ゲーのボタン練習アプリです。
- ランダムでお題が表示され、指定されたボタンを押すことでスコアが得られます。
- Windowsで動作し、ストリートファイター6モダンをはじめ、様々な格闘ゲームの練習に活用できます。

## 技術スタック
- フロントエンド: 特定のGUIフレームワークは明記されていませんが、PythonとWindowsのAPIを利用してユーザーインターフェースが構築されています。ボタン表示やウィンドウの最前面・最背面化機能などが含まれます。
- 音楽・オーディオ: 音楽・オーディオに関する特定の技術は情報に含まれていません。
- 開発ツール: Python (主要プログラミング言語)、pip (Pythonパッケージ管理)、Visual Studio Code (推奨エディタ)、Git (バージョン管理システム)、Pylint (Python静的コード分析ツール)。
- テスト: テストフレームワークは明記されていませんが、`tests/` ディレクトリ内に単体テストコードが含まれており、アプリケーションの動作検証に用いられています。
- ビルドツール: TOML (設定ファイルの記述形式)、`requirements.txt` (Pythonの依存パッケージ管理)、`button_challenge.bat` (Windows向け実行スクリプト)。ドキュメント生成にはJekyllが使用されている可能性があります。
- 言語機能: Python (アプリケーション開発に用いられる高水準プログラミング言語)。
- 自動化・CI/CD: GitHub Actions (リポジトリのREADMEファイルの自動翻訳などのワークフローに利用)。
- 開発標準: .editorconfig (複数開発者間でのコードスタイル統一)、.pylintrc (Pythonコードの品質と一貫性を保つためのPylint設定)。
